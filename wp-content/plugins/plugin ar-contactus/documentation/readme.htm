Contact us all-in-one button with callback request feature

Facebook Messenger, WhatsApp, Skype, Slack, Viber, Telegram, Call Back, Email and more channels just in one button! Let customers choose their favorite method to reach you with 1 click and follow up the conversation anytime and anywhere!

How it works

This plugin displays animated "contact-us" button on every page of your WordPress site.

You can customize button position, button size, button color, menu size, button animation speed (please see configuration section to learn how to do this).

If "prompt" feature is enabled, prompt messages will be shown to visitors.

When visitor clicks on "contact-us" button menu will be shown.

Then visitor can choose their favorite method to reach you.

Each of the menu items (except callback one) its just a link to a propely messanger or social network. So to give visitors opportunety contact you via FaceBook Messenger you can add link like https://m.me/areamaDevelopment replacing areamaDevelopment to a your correct FaceBook page.

When visitor clicks on "Callback request" callback form will be opened. When visitor submit callback form, admins will receive email (if "send email" feature is active) and this request will be displayed in "Callbacks" menu.

You can customize all icons, texts, colors from plugin settings in your WordPress admin panel.

Plugin configuration

General configuration

  • Enable on mobile - you can enable/disable widget on mobile devices using this option.
  • Disable widget on pages - you can disable widget on several pages using this option. For example: if you want to disable widget on home page of your site type / in this field. If you want to disable widget on page https://yoursite.com/2018/06/01/hello-world/ type /2018/06/01/hello-world/ to this field.

Button settings

  • Color theme - set color of the "contact-us" button and callback request form. We are recomends you to use colors that contrast with white color.
  • Button size - size of the main button. You can choose Large, Medium or Small.
  • Position - set "contact-us" button position. You can choose left or right.
  • X-axis offset - set x-axis "contact-us" button offset.
  • Y-axis offset - set y-axis "contact-us" button offset.
  • Pulsate speed - "contact-us" button pulsate speed in ms.
  • Icon slider speed - icons change animation speed. You can set this option to 0 to disable button animation.
  • Text - text on "contact-us" button.
  • Enable button drag - if "on" "contact-us" button will be allowed to drag.

Menu settings

  • Menu size - size of menu. You can choose between Large or Small.
  • Background color - background color of the menu.
  • Text color - text color of the menu item.
  • Hovered item background color - background color of the hovered menu item.
  • Hovered item text color - text color of the hovered menu item.

Callback popup settings

  • Timeout - if positive then countdown timer will be displayed when visitor submit callback form:
  • Message - callback form text.
  • Phone field placeholder - placeholder for phone field.
  • Proccess message - this message will be displayed if timeout is positive.
  • Success message - this message will be displayed when callback request is sent correctly.
  • Fail message - will be displayed if something goes wrong while sending callback request.
  • Button title - callback form submit button title.
  • Send email - if enabled, email will be send when visitor request a callback.
  • Email list - list of email addresses for sending email notifications about callback requests.
  • Enable Twilio Integration - This module can be integrated with Twilio SMS service.
    To use this integration you need to signup to Twilio. Please follow this link:
    https://www.twilio.com/try-twilio
  • Twilio API Key - Twilio account SID. You can find it in Twilio console https://www.twilio.com/console
  • Twilio Auth Token - Twilio auth token. You can find it in Twilio console https://www.twilio.com/console
  • Twilio phone - Twilio phone number. You can find it in Twilio console https://www.twilio.com/console
  • Send SMS to this phone - you will receive SMS to this phone number.
  • SMS text - text of SMS. {phone} token will be replaced to phone entered in callback request form.
  • Integrate with Google reCaptcha - This module can be integrated with Google reCaptcha to prevent bots from sending callback requests.
    To use this integration you need to get reCaptcha V3 key and secret. Please follow this link:
    https://g.co/recaptcha/v3
  • Google reCaptcha Site Key
  • Google reCaptcha Secret
  • Hide Google reCaptcha logo - if you using reCaptcha integration, you will see reCaptcha logo at the right bottom corner of your site. If you want hide this logo you can enable this option.

Prompt settings

  • Enable - enable "prompt" feature.
  • Delay first message - delay for first prompt message (in ms).
  • Loop messages - loop all prompt messsages.
  • Close last messages - close last prompt message.
  • Typing time - animated typing simulation time (in ms). Set to 0 to disable typing simulation.
  • Message time - message display time (in ms).

Menu items

In this section you can configure menu items you want to display.

After plugin installation default menu items will be added to this table. You can edit or remove any of them.

To add new menu item click Add button at the top of the page and fill out the fields:

Link field can be a link to properly social network or messenger. Or it can be callback - to display callback form by clicking this menu item.

You can use Javascript field to write custom click handler. Please see our jquery plugin documentation for more details.

By default new menu item will be added to the end of the menu items list. You can change item position by dragging it via drag handle.

Callback requests

You can see all callback requests in "Callbacks" section of the main menu. You can change status off each callback request.
